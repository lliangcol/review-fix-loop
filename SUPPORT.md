# Support

Use the right channel so maintainers can respond with the needed context.

## Bugs

Open a bug report with:

- operating system;
- Python version;
- exact command;
- adapter config path;
- snapshot or gate mode;
- minimal reproduction.

## Usage Questions

Open a question issue or GitHub Discussion when enabled. Include the workflow you are trying to run and what the agent should do after each pass.

## Feature Ideas

Open a feature request with the stale-context problem it solves, the expected local workflow, and any adapter impact.

## Security

Do not open public issues for secrets, private source, or vulnerability details. Follow `SECURITY.md`.
